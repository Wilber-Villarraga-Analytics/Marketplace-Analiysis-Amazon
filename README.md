# Análisis de Oportunidades en Amazon: El Caso de los Jeans Levi's 

Este proyecto nació de una pregunta simple pero poderosa: ¿Es posible encontrar nichos de mercado ocultos en medio de miles de productos parecidos? 

A través de un análisis exhaustivo de más de 700 registros de la categoría "Clothing, Shoes & Jewelry", identifiqué patrones de demanda y competencia.

### Hallazgos Clave:
- **Resiliencia en la Limpieza:** Realicé múltiples iteraciones de limpieza de datos para asegurar que los "outliers" no fueran errores, sino oportunidades reales.
- **Detección de Patrones:** Identifiqué que ciertos modelos de Jeans Levi's presentan una relación inusual entre precio y volumen de reseñas, señalando una alta demanda con baja saturación de competencia.
- **Herramientas:** Python (Pandas, Matplotlib, Seaborn, Scipy).

"La estadística me permitió ver lo que el ojo humano ignora entre 700 datos: una oportunidad de negocio tangible."
